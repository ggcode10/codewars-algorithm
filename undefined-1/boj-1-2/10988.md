# 펠린드롬인지 확인하기 - 10988

펠린드롬이란 앞으로 읽을 때와 거꾸로 읽을 때 똑같은 단어를 말합니다.

## 1. 접근

인풋으로 받는 String과 그 인풋을 reverse한 값이 같으면 펠린드롬입니다.

```swift
var input = readLine()!

// 인풋이랑, 인풋 리버스랑 비교해서 같으면 펠린드롬

var newStr = String(input.reversed())

if input == newStr {
    print("1")
} else {
    print("0")
}

```
