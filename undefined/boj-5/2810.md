# 컵홀더 - 2810

Foundation을 import해야 사용할 수 있는, replacingOccurrences를 사용합니다.

LL을 L로 바꿔준 후, 인풋으로 들어온 좌석 수와 seat.count를 비교합니다. 좌석수가 사람수와 같거나 작으면 그대로 좌석수를 리턴,

반대라면 사람의 수를 리턴합니다.

```swift
// Some code
import Foundation

var N = Int(readLine()!)!
var seat = readLine()!.replacingOccurrences(of: "LL", with: "L")

if N <= seat.count + 1 {
    print(N)
}
else {
    print(seat.count + 1)
}

```
